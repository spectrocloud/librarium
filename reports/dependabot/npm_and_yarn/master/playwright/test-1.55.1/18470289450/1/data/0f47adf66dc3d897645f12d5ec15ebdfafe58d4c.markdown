              - paragraph [ref=e288]: VerteX allows tenants to use AWS Secret Token Service (STS) to add AWS cloud accounts to their VerteX environment. In order to do this, VerteX relies on another AWS cloud account that is configured on the VerteX instance to assume a custom role created on the tenant's AWS account.
              - paragraph [ref=e289]:
                - text: This custom role establishes a trust relationship between the tenant's AWS account and the AWS account bound to the VerteX instance. It allows the VerteX-bound AWS account to receive temporary credentials from the tenant's AWS account to deploy infrastructure in AWS using STS. For more information on how to add an AWS cloud account to a tenant using STS, refer to the
                - link "Add an AWS Account to Palette" [ref=e290] [cursor=pointer]:

                - text: guide.
