Follow the steps below to create a Palette API key. This is required for the <VersionedLink text="Cluster Management with Terraform" url="getting-started/aws/deploy-manage-k8s-cluster-tf"/>
tutorial.

1. Log in to [Palette](https://console.spectrocloud.com) as a tenant admin.

2. Switch to the **Tenant Admin** scope

3. From the left **Main Menu**, select **Tenant Settings**.

4. From the **Tenant Settings Menu**, select **API Keys**.

5. Click **Add New API key**.

6. Fill out the following input fields and click **Generate**.

| **Input Field**     | **Description**                                                                                                   |
| ------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **API Key Name**    | Assign a name to the API key.                                                                                     |
| **Description**     | Provide a description for the API key.                                                                            |
| **User Name**       | Select the user to assign the API key.                                                                            |
| **Expiration Date** | Select an expiration date from the available options. You can also specify a custom date by selecting **Custom**. |

7. Copy the API key and save it in a secure location, such as a password manager.

:::warning

Keep the API key in a secure location, as you will not be able to view it again.

:::
