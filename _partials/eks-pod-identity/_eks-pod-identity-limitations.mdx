- This authentication option is only available for [self-hosted Palette](../../../../self-hosted-setup/palette/) or [Palette VerteX](../../../../self-hosted-setup/vertex/) instances deployed on Amazon EKS clusters.

- Only Amazon EKS clusters can be deployed as workload clusters when using EKS Pod Identity for authentication.
