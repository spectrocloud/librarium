The Palette MCP server contains the following tools that you can leverage for your Palette operations:

- `gather_or_delete_clusterprofiles` - Fetches information about or deletes
  [cluster profiles](/profiles/cluster-profiles).
- `gather_or_delete_clusters` - Fetches information about or deletes [clusters](/clusters).
- `getKubeconfig` - Downloads cluster [kubeconfig](/clusters/cluster-management/kubeconfig) files.
- `search_and_manage_resource_tags` - Manages resource tags for cluster profiles, clusters,
  [Edge hosts](/clusters/edge), [cluster templates](/cluster-templates), and
  [cluster templates policies](/cluster-templates/create-cluster-template-policies).
