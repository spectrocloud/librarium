Use the following procedure to verify that your Azure cloud account has been added in {props.edition}.

1. Log in to {props.edition} as a tenant admin.

2. From the left main menu, select **Tenant Settings**.

3. From the **Tenant Settings Menu**, select **Cloud Accounts**.

4. Confirm that the {props.cloud_account_actual} cloud account is listed under **Azure** with all other available Azure cloud accounts.
