Due to upstream issues with Kairos, Unified Kernel Image (UKI)-based Trusted Boot images are pinned to Kairos v3.5.9. This version is used whenever `IS_UKI=true` is set in the `.arg` file. Refer to Kairos GitHub issues [3831](https://github.com/kairos-io/kairos/issues/3831) and [4046](https://github.com/kairos-io/kairos/issues/4046) for more information.
