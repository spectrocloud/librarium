# Containers

Small digital units in which parts of software called apps are held, processed and computed.

# cURL

[cURL](https://en.wikipedia.org/wiki/CURL#cURL) is a command-line tool for getting or sending data including files using URL syntax. Use cURL to download physical web pages, images, documents, and files.

# Docker

Docker is a solution that allows the conversion of software into units called containers.
